# Sobre

Este pacote foi processado para priorizar conteúdo. As páginas HTML foram geradas a partir dos arquivos Markdown presentes no pacote original. A interface foi criada após gerar o conteúdo.

## Instruções

- Extraia este zip.
- Abra index.html no navegador.
- Para alterar textos edite os arquivos .md originais dentro do pacote extraído e substitua as páginas ou reexecute o processo.
