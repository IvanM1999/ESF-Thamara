Lógica de Programação

A lógica de programação é a base de toda a computação moderna. Ela permite estruturar pensamentos, organizar problemas e criar soluções que um computador consegue executar.

Este curso é voltado a iniciantes, estudantes, servidores públicos e qualquer pessoa que deseja compreender como sistemas computacionais funcionam.

Objetivos

  • Compreender o que é um algoritmo
  • Organizar soluções de forma lógica
  • Utilizar variáveis e operadores
  • Criar estruturas de decisão
  • Aplicar estruturas de repetição
  • Entender a lógica de qualquer linguagem de programação

Conteúdo

1. Pensamento lógico

Pensar logicamente significa organizar ideias em uma sequência clara de passos.

Exemplo:

1. Olhar para a esquerda  
2. Olhar para a direita  
3. Se não houver carros, atravessar

2. Algoritmos

Algoritmo é uma sequência finita de instruções para resolver um problema. 

Fazer café:
1. Colocar água
2. Adicionar café
3. Ligar a cafeteira
4. Esperar o café ficar pronto

3. Variáveis

Variáveis são espaços de memória usados para armazenar dados.

  • idade = 25
  • nome = "Maria"
  • salario = 3200
